# installer
Installers for any software


# Run default recommendation installation
```
cd <product>
./run
```

# Get help on all steps available
```
./run help


  Usage: ./run action [action action ...]

  Actions include:

	- install
	- version
	- configure
	- start
	- cleanup

	- stop
	- remove
	- purge
	- help

```

# Perform script debugging
```
TRACE=y ./run
```
